# MVP Screens

## 1) Onboarding
- Choose 1–2 goals (Skin / Gut / Pregnancy prep)
- Toggle Gentle Mode (no tiers, just reflections)
- Optional dietary preferences

## 2) Today (Home)
- Log a meal button
- Goal cards with category tiles:
  - Status: Not yet / Growing / Strong (hidden if Gentle Mode)
- One “Add-One” suggestion

## 3) Log Meal
- Select meal type
- Search foods from curated list
- Add foods (optional Small/Medium/Large)
- Save meal
- Optional: Repeat last meal, Recent foods

## 4) Food Detail / Category Detail
- What this supports (tags)
- Why it matters for selected goals
- Easy pairings (1–2)

## 5) Insights
- Weekly recap (gentle language)
- 1–2 suggestions for next week
- Optional symptom tracking + light correlations
